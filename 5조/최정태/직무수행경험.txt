직무수행경험



그린아카데미의 6개월간의 교육과정에서 했던 프로젝트를 수행한 경험을 기반으로 팀원들과 공유하며 프로젝트 수행에 주도적으로 참여하였습니다.
프론트엔드 교육과정에서는 기능 개발을 위주로 배우고자 오늘의 집이라는 사이트를 선정하였고, 제가 맡은 역할은 상세페이지입니다.
MySQL로 테이블 구조를 미리 파악하여 테이블 생성과 쿼리문 작성 등 데이터를 서버와 연동하였고, Node와 Express로 서버 작업을 하였습니다. 
서버는 MVC패턴으로 기능 별로 분리 시켜주었고, GitHub, Git GUI(Source Tree)를 사용하여 협업하였습니다. 
첫번째 프로젝트를 마치고 느꼈던 점은, 
리덕스를 사용하지 않아 Deps가 깊어지는 문제와 효율적인 상태관리와 유지보수가 어렵다는 점과 css가 너무 길어진다는 문제였습니다.
해당 부분을 보완하고자 두번째 프로젝트에서는 리덕스를 사용하고, Custom Hook을 더 활용함으로써 효율적인 관리를 할 수 있었습니다.
Css는 Module.css와 sass를 같이 사용하여 css가 길어지고 유지보수가 어려워지는 문제를 해결하였습니다.
두번째 프로젝트에서 맡은 역할은 로그인과 회원가입, 로그아웃, 탈퇴 등입니다.
프론트 단에서 유효성검사를 제대로 진행 해주어야 된다는 내용을 듣고, 유효성 검사를 꼼꼼히 진행하였습니다.
로그인 회원가입을 맡으면서 회원 인증을 하는 각 방식의 장점과 단점을 공부 할 수 있었고, 
비밀번호 암호화, 쿠키와 세션, 스토리지, 토큰 인증 방식, 이메일인증, 로봇체크, 우편번호 API, 정규식등 많은 것을 배울 수 있었습니다.

앞으로도 이렇게 다양한 기능들과 언어, 다양한 방식을 접하고 경험해보고 싶은 생각이 들었습니다.
배우고자하는 기술 중 TypeScript와 테일윈드, Next.js, java 등 다양한 언어와 기술을 습득하고 싶다는 생각이 들었습니다.
