영화사이트 제작

1. tmdb 사이트에서 api 키를 받아서 영화사이트를 제작
2. 자바스크립트 모듈을 사용해서 제작할 것
  2-1 모듈은 export,import 방식
3. fetch를 사용해서 영화사이트의 정보들을 가져올 것
  3-1 fetch안에 활용할 api주소를 넣기
    3-1-1 import해서 가져온 변수들을 주소에 넣는 방법
      3-1-1-1 주소에 넣고 사용할 함수를 만든다. 함수를 만들어두고 필요할 때 사용할 것
      3-1-1-1-1 함수를 만들 때도 export를 사용해서 만든다. 이 때 사용할 때 넣을 매개변수값을 지정해줄 수 있다.
      3-1-1-1-2 영화 데이터를 가져오고나서 만들어야하기 때문에 비동기(프로미스)로 만들어준다.
      3-1-1-1-2-1 비동기(프로미스)로 만들고 await를 사용할 때는 async를 넣어주어야 한다.
      3-1-1-1-2-2 실행할 함수들을 만든다.
      3-1-1-1-2-2-1 import된 변수들을 사용해서 만드는데, await fetch를 사용해서 가져온다.
      3-1-1-1-2-2-2 fetch로 가져온 파일들을 json으로 바꿔주고 다시 저장해준다.
      3-1-1-1-2-2-2 resolve해서 데이터를 반환시켜준다.

    3-2-1 가져온 데이터를 함수로 만들어준 후 호출해서 사용할 때
      3-2-1-1 export해놓은 데이터들을 import로 가져온다.
      3-2-2-1 헤더 > 비쥬얼 > 콘텐츠 > 푸터 순으로 데이터를 순서대로 받아오게 작업해주어야한다. 
      3-2-2-1-1  순서대로 처리되게끔 비동기로 작업하려면 비동기(프로미스)를 사용해주어야한다.
      3-2-2-1-1-1  사용할 함수를 변수로 가져온다. 예 : let movieData = await getMovies(options.playing)
      3-2-2-1-1-1-1 객체로 만들어놓은 options의 playing은 현재상영작을 사용하겠다는 뜻이다.   
      3-2-2-1-1-1-1-1 이때 함수를 가져와서 사용하려면 await을 넣어야 사용할 수 있다.
      3-2-2-1-1-2  현재상영작의 정보를 가져왔다면 현재상영작 정보 안에있는 추가 정보들 중에서 results를 가져와서 사용한다.
      3-2-2-1-1-2-1  20개가 넘는 데이터에서 5개만 사용한다. 
      3-2-2-1-1-2-1-1  가져온 results를 slice를 활용해서 5개로 쪼개준후에 그 값을 사용한다.
      3-2-2-1-1-2-1-2  movies = movies.slice(0, 5)
      3-2-2-1-2 5개로 쪼개진 results 값을 for of를 이용해서 각각 실행해준다.
      3-2-2-1-2-1 filter,map,forEach는 함수안에 await를 넣으면 사용이 불가해서 for of나 for in을 사용하는데, 배열이라 for of를 사용했다.
      3-2-2-1-3  results 안에있는 값들을 활용해서 이미지를 출력한다.
      3-2-2-1-3-1  results 안에는 각 영화정보들의 id값,이미지설명.jpg, 제목, 설명 등 다양한 정보들을 가지고있다.
      3-2-2-1-3-2 한글로된 리뷰가 없는 경우가 종종있어서 이럴 경우에 대안법으로 아래에 한글정보가 없을 때
      3-2-2-1-3-2-1 영어 데이터를 찾고,영어로라도 출력되게끔 만들어주었다.
      
       

  



      3-2-2-1-2 함수들을 await를 사용해서 순서대로 실행할 수 있게 만들어줄 수 있다.




  3-2 하나의 api가 아닌 여러개의 api들을 가져와야 하기 때문에 변수를 사용할것
    3-2-1 주소를 분리해서 여러개의 변수를 만든다.
    3-2-2 변하는 api정보들을 객체로 만들어준다.
      3-2-2-1 만들어준 변수들을 import해서 사용해준다.
        3-2-2-1-1 import 사용방법
        3-2-2-1-1 1 : export를 해서 만들어준 변수(값)들을 구조분해해서 저장해준다.
        3-2-2-1-1 1 : from을 사용해서 모듈의 주소를 가져온다.

    

4. api를 가져오는 동안 출력이 되면 안되기 때문에 프로미스를 사용해서 비동기로 처리할것
  4-1 헤더 > 비쥬얼 > 콘텐츠 순으로 구조를 만들때 순서대로 헤더,비쥬얼 작업이 끝나면 콘텐츠 작업을 시작할 것
    4-1-1 데이터를 가져올 때 순서를 지정하는 방법 구현한 각 기능들에게 await를 넣어야함
